# Fonts folder – Nexokora Techno & Nohemi

Beberapa font dipakai dari folder ini. Letakkan file yang sesuai agar tampilan sesuai desain.

---

## Nexokora Techno (logo X-Arena)

Agar teks **X-Arena** memakai font **Nexokora Techno** (landing page & sidebar), taruh file font di folder ini.

1. **Unduh font Nexokora Techno**  
   - Sumber resmi: [Nirmana Visual – Nexokora Techno](https://nirmanavisual.com/product/nexokora-techno/)  
   - Atau dari sumber lain yang menyediakan file `.ttf` / `.otf`.

2. **Nama file di folder ini:**
   - `NexokoraTechno-Regular.woff2` (prioritas, untuk web)
   - **atau** `NexokoraTechno-Regular.ttf`

Tanpa file di atas, browser memakai fallback **Orbitron**.

---

## Nohemi (greeting di top bar header)

Agar greeting di header memakai font **Nohemi**, taruh file font di folder ini.

1. **Unduh font Nohemi** (gratis, berbagai weight)  
   - Cari "Nohemi font" dari sumber tepercaya (mis. FreeFontDL, Fonts Hut, atau distributor resmi).

2. **Nama file di folder ini:**
   - `Nohemi-Regular.woff2` atau `Nohemi-Regular.woff` atau `Nohemi-Regular.ttf`

Tanpa file di atas, browser memakai fallback **sans-serif**.

---

**Restart dev server** (`npm run dev`) lalu refresh browser setelah menambah file font.
